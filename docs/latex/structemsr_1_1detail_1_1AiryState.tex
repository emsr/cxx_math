\subsubsection*{template$<$typename Tp$>$\newline
struct emsr\+::detail\+::\+Airy\+State$<$ Tp $>$}

This struct defines the \mbox{\hyperlink{classemsr_1_1detail_1_1Airy}{Airy}} function state with two presumably numerically useful \mbox{\hyperlink{classemsr_1_1detail_1_1Airy}{Airy}} functions and their derivatives. The data mambers are directly accessible. The lone method computes the Wronskian from the stored functions. A static method returns the correct Wronskian.
