134  *
135  * The spherical Legendre function is defined by
136  * @f[

148  * @param l The order <tt> l >= 0 </tt>
149  * @param m The degree <tt> m >= 0 </tt> and <tt> m <= l </tt>
150  * @param theta The radian polar angle argument
151  * @see assoc_legendre for the unnormalized associated Legendre polynomial.

183  /**
184  * Return the Legendre function of the second kind @f$ Q_l(x) @f$ of
185  * nonnegative degree @c l and real argument @f$ |x| <= 0 @f$.
186  *
187  * The Legendre function of the second kind of order @c l
188  * and argument @c x, @f$ Q_l(x) @f$, is defined by:
189  * @f[
190  * Q_l(x) = \frac{1}{2} \log{\frac{x+1}{x-1}} P_l(x)
191  * - \sum_{k=0}^{l-1}\frac{(l+k)!}{(l-k)!(k!)^2 s^k}
192  * \left[\psi(l+1) - \psi(k+1)\right](x-1)^k
193  * @f]
194  * where @f$ P_l(x) @f$ is the Legendre polynomial of degree @c l
195  * and @f$ \psi(x) @f$ is the digamma or psi function which for integral
196  * argument is related to the harmonic number:
197  * @f$ \psi(n) = -\gamma_E + H_n @f$.
198  *
199  * @tparam Tp The floating-point type of the argument @c x.
200  * @param l The degree @f$ l >= 0 @f$
201  * @param x The argument @c abs(x) <= 1
202  * @throw std::domain_error if @c abs(x) > 1
203  */
