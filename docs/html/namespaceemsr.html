enum emsr::gauss_quad_type

Enumeration for differing types of Gauss quadrature. The gauss_quad_type is used to determine the boundary condition modifications applied to orthogonal polynomials for quadrature rules.

Enumerator

Gauss quadrature.

Gauss-Radau quadrature including the node -1.
