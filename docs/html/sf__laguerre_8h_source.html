14  * of nonnegative degree @c n, nonnegative order @c m
15  * and real argument @c x.
16  *
17  * The associated Laguerre function of real order @f$ \alpha @f$,
18  * @f$ L_n^{(\alpha)}(x) @f$, is defined by
19  * @f[
20  * L_n^{(\alpha)}(x) = \frac{(\alpha + 1)_n}{n!}
21  * {}_1F_1(-n; \alpha + 1; x)
22  * @f]
23  * where @f$ (\alpha)_n @f$ is the Pochhammer symbol and
24  * @f$ {}_1F_1(a; c; x) @f$ is the confluent hypergeometric function.
25  *
26  * The associated Laguerre polynomial is defined for integral
27  * order @f$ \alpha = m @f$ by:
28  * @f[
29  * L_n^{(m)}(x) = (-1)^m \frac{d^m}{dx^m} L_{n + m}(x)
30  * @f]
31  * where the Laguerre polynomial is defined by:
32  * @f[
33  * L_n(x) = \frac{e^x}{n!} \frac{d^n}{dx^n} (x^ne^{-x})
34  * @f]
35  * and @f$ x >= 0 @f$.
36  * @see laguerre for details of the Laguerre function of degree @c n
37  *
38  * @tparam Tp The floating-point type of the argument @c x.
39  * @param n The degree of the Laguerre function, <tt>n >= 0</tt>.
40  * @param m The order of the Laguerre function, <tt>m >= 0</tt>.
41  * @param x The argument of the Laguerre function, <tt>x >= 0</tt>.

43  */
