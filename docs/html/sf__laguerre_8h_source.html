33  /**
34  * Return the associated Laguerre polynomial @f$ L_n^{(m)}(x) @f$
35  * of nonnegative degree @c n, nonnegative order @c m
36  * and real argument @c x.
37  *
38  * The associated Laguerre function of real order @f$ \alpha @f$,
39  * @f$ L_n^{(\alpha)}(x) @f$, is defined by
40  * @f[
41  * L_n^{(\alpha)}(x) = \frac{(\alpha + 1)_n}{n!}
42  * {}_1F_1(-n; \alpha + 1; x)
43  * @f]
44  * where @f$ (\alpha)_n @f$ is the Pochhammer symbol and
45  * @f$ {}_1F_1(a; c; x) @f$ is the confluent hypergeometric function.
46  *
47  * The associated Laguerre polynomial is defined for integral
48  * order @f$ \alpha = m @f$ by:
49  * @f[
50  * L_n^{(m)}(x) = (-1)^m \frac{d^m}{dx^m} L_{n + m}(x)
51  * @f]
52  * where the Laguerre polynomial is defined by:
53  * @f[
54  * L_n(x) = \frac{e^x}{n!} \frac{d^n}{dx^n} (x^ne^{-x})
55  * @f]
56  * and @f$ x >= 0 @f$.
57  * @see laguerre for details of the Laguerre function of degree @c n
58  *
59  * @tparam Tp The floating-point type of the argument @c x.
60  * @param n The degree of the Laguerre function, <tt>n >= 0</tt>.
61  * @param m The order of the Laguerre function, <tt>m >= 0</tt>.
62  * @param x The argument of the Laguerre function, <tt>x >= 0</tt>.
63  * @throw std::domain_error if <tt>x < 0</tt>.
64  */
