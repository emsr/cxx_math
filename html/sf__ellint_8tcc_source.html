62  * @brief Return the Carlson elliptic function
63  * @f$ R_C(x,y) = R_F(x,y,y) @f$ where @f$ R_F(x,y,z) @f$
64  * is the Carlson elliptic function of the first kind.
65  *
66  * The Carlson elliptic function is defined by:
67  * @f[
68  * R_C(x,y) = \frac{1}{2} \int_0^\infty
69  * \frac{dt}{(t + x)^{1/2}(t + y)}
70  * @f]
71  *
72  * Based on Carlson's algorithms:
73  * - B. C. Carlson Numer. Math. 33, 1 (1979)
74  * - B. C. Carlson, Special Functions of Applied Mathematics (1977)
75  * - Numerical Recipes in C, 2nd ed, pp. 261-269,
76  * by Press, Teukolsky, Vetterling, Flannery (1992)
77  *
78  * @param __x The first argument.
79  * @param __y The second argument.
80  * @return The Carlson elliptic function.
81  */

95  return _S_NaN;

106  }
107  else

110  auto __yt = __y;
