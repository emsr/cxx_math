231  * @section general General Features
232  *
233  * @subsection promotion Argument Promotion
234  * The arguments suppled to the non-suffixed functions will be promoted
235  * according to the following rules:
236  * 1. If any argument intended to be floating point is given an integral value
237  * That integral value is promoted to double.
238  * 2. All floating point arguments are promoted up to the largest floating
239  * point precision among them.
240  *
241  * @subsection NaN NaN Arguments
242  * If any of the floating point arguments supplied to these functions is
243  * invalid or NaN (std::numeric_limits<Tp>::quiet_NaN),
244  * the value NaN is returned.
245  *
246  * @section impl Implementation
247  *
248  * We strive to implement the underlying math with type generic algorithms
249  * to the greatest extent possible. In practice, the functions are thin
250  * wrappers that dispatch to function templates. Type dependence is
251  * controlled with std::numeric_limits and functions thereof.
252  *
253  * We don't promote @c float to @c double or @c double to <tt>long double</tt>
254  * reflexively. The goal is for @c float functions to operate more quickly,
255  * at the cost of @c float accuracy and possibly a smaller domain of validity.
256  * Similaryly, <tt>long double</tt> should give you more dynamic range
257  * and slightly more pecision than @c double on many systems.
258  *
259  * @section testing Testing
260  *
261  * These functions have been tested against equivalent implementations
262  * from the <a href="http://www.gnu.org/software/gsl">
263  * Gnu Scientific Library, GSL</a> and

338  /**
339  * Return the associated Laguerre polynomial @f$ L_n^m(x) @f$
340  * of nonnegative order @f$ n @f$, nonnegative degree @f$ m @f$
341  * and real argument @f$ x @f$.
342  *
343  * The associated Laguerre function of real degree @f$ \alpha @f$,
344  * @f$ L_n^\alpha(x) @f$, is defined by
345  * @f[
346  * L_n^\alpha(x) = \frac{(\alpha + 1)_n}{n!}
347  * {}_1F_1(-n; \alpha + 1; x)
348  * @f]
349  * where @f$ (\alpha)_n @f$ is the Pochhammer symbol and
350  * @f$ {}_1F_1(a; c; x) @f$ is the confluent hypergeometric function.
351  *
352  * The associated Laguerre polynomial is defined for integral
353  * degree @f$ \alpha = m @f$ by:
354  * @f[
355  * L_n^m(x) = (-1)^m \frac{d^m}{dx^m} L_{n + m}(x)
356  * @f]
357  * where the Laguerre polynomial is defined by:
358  * @f[
359  * L_n(x) = \frac{e^x}{n!} \frac{d^n}{dx^n} (x^ne^{-x})
360  * @f]
361  * and @f$ x >= 0 @f$.
362  * @see laguerre for details of the Laguerre function of degree @c n
363  *
364  * @tparam _Tp The floating-point type of the argument @c __x.
365  * @param __n The order of the Laguerre function, <tt>__n >= 0</tt>.
366  * @param __m The degree of the Laguerre function, <tt>__m >= 0</tt>.
367  * @param __x The argument of the Laguerre function, <tt>__x >= 0</tt>.
368  * @throw std::domain_error if <tt>__x < 0</tt>.
369  */
370  template<typename _Tp>
371  inline typename __gnu_cxx::__promote<_Tp>::__type
372  assoc_laguerre(unsigned int __n, unsigned int __m, _Tp __x)
373  {
374  typedef typename __gnu_cxx::__promote<_Tp>::__type __type;
375  return __detail::__assoc_laguerre<__type>(__n, __m, __x);
376  }

447  /**
448  * Return the beta function, @f$B(a,b)@f$, for real parameters @c a, @c b.
449  *
450  * The beta function is defined by
451  * @f[
452  * B(a,b) = \int_0^1 t^{a - 1} (1 - t)^{b - 1} dt
453  * = \frac{\Gamma(a)\Gamma(b)}{\Gamma(a+b)}
454  * @f]
455  * where @f$ a > 0 @f$ and @f$ b > 0 @f$
456  *
457  * @tparam _Tpa The floating-point type of the parameter @c __a.
458  * @tparam _Tpb The floating-point type of the parameter @c __b.
