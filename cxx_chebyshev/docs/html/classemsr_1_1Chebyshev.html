template<typename Tp>
class emsr::Chebyshev< Tp >

_Chebyshev represents a Chebyshev fit of a function. Given a function func, the lower limit a, the upper limit b, and the number of points n _Chebyshev contains the coefficients of a Chebyshev polynomial expansion such that

Use these constructors with moderately large n of 30 or 50. Then truncate the Chebyshev fit to a smaller number of terms to satisfy the accuracy requirements.

Constructs a Chebyshev expansion from a polynomial.

A polynomial valid in the range x=a to x=b is specified by coefficients d[0..m-1]. An equivalent array of Chebyshev coefficients, c[0..m-1], is output. The index mfew will be set to the index of the first nonzero Chebyshev coefficient smaller than err. Then the polynomial is approximated by the first mfew cefficients c[0..mfew-1].
