\subsubsection*{template$<$typename Tp$>$\newline
class emsr\+::\+Chebyshev$<$ Tp $>$}

\+\_\+\+Chebyshev represents a \mbox{\hyperlink{classemsr_1_1Chebyshev}{Chebyshev}} fit of a function. Given a function {\ttfamily func}, the lower limit {\ttfamily a}, the upper limit {\ttfamily b}, and the number of points {\ttfamily n} \+\_\+\+Chebyshev contains the coefficients of a \mbox{\hyperlink{classemsr_1_1Chebyshev}{Chebyshev}} polynomial expansion such that 

\[ f(x) = k\sum_{0}^{n-1} c_k T_k(x) - c_0/2. \] Use these constructors with moderately large n of 30 or 50. Then truncate the \mbox{\hyperlink{classemsr_1_1Chebyshev}{Chebyshev}} fit to a smaller number of terms to satisfy the accuracy requirements.

Constructs a \mbox{\hyperlink{classemsr_1_1Chebyshev}{Chebyshev}} expansion from a polynomial.

A polynomial valid in the range x=a to x=b is specified by coefficients d\mbox{[}0..m-\/1\mbox{]}. An equivalent array of \mbox{\hyperlink{classemsr_1_1Chebyshev}{Chebyshev}} coefficients, c\mbox{[}0..m-\/1\mbox{]}, is output. The index mfew will be set to the index of the first nonzero \mbox{\hyperlink{classemsr_1_1Chebyshev}{Chebyshev}} coefficient smaller than err. Then the polynomial is approximated by the first mfew cefficients c\mbox{[}0..mfew-\/1\mbox{]}.
