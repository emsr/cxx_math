template<typename Tp, typename AFun, typename BFun, typename TailFun>
class emsr::LentzContinuedFraction< Tp, AFun, BFun, TailFun >

A modified Lentz continued fraction evaluator. This class requires three functors: A partial numerator function A partial denominator function A tail function
